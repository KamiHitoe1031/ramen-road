# 🍜 らーめん道 ～至高の一杯～

ドラフト式対戦ボードゲーム。3-4人のプレイヤーがオンラインで具材を取り合い、3×3の丼に盛り付けて、味と芸術点で勝負！

## 遊び方

1. 部屋を作って友達を招待（3-4人）
2. ラーメン屋キャラを選ぶ
3. スープと麺を選ぶ
4. 具材ドラフト（寿司ゴー方式で取り合い！）
5. 3×3の丼に盛り付け
6. 4つの基準で採点→称号授与→勝者決定！

## 開発

```bash
# ローカル開発（Phase 1-2: クライアントのみ）
npx http-server public -p 8080 -c-1

# サーバー込み開発（Phase 3以降）
npm install
npm run dev
```

## デプロイ

Railway にて自動デプロイ（mainブランチへのpushで発動）

## 技術スタック

- Phaser 3（ゲームエンジン）
- Node.js + Express + Socket.io（サーバー）
- Railway（ホスティング）
